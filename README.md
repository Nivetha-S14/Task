\# 🔐 Full Stack User Authentication System



\## 📌 Project Overview



This is a full-stack web application that implements a secure user authentication system using React for the frontend, Python for the backend, and a database for storing user credentials.



The system allows only registered users to access the application. Unregistered users are denied access.



\---



\## ⚙️ Tech Stack



\### Frontend

\- React (Vite)



\### Backend

\- Python

\- FastAPI 



\### Database

\- MySQL 



\---



\## 🚀 Features

\- User Registration system

\- Secure Login authentication

\- Database validation for users

\- Protected access control (deny unauthorized users)

\- Frontend–Backend API integration

\- Clean separation of concerns (Frontend / Backend)
